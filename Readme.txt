-Import database và chạy project (sử dụng MongooseDB):

	+Copy thư mục database trong file đồ án vào ổ C

	+Mở cửa sổ cmd của window, sau đó trỏ đến thư mục bin của mongodb bằng lệnh "cd", với phiên bản 5.0 thường là:
	"C:\Program Files\MongoDB\Tools\100\bin"

	+Gõ lệnh: "mongorestore -d finallProject C:\database\finallProject"

	+Mở project đồ án

	+Ở cửa sổ terminal viết lệnh "npm i"

	+Ở cửa sổ terminal viết lệnh "npm start"
